Personal Portfolio Website

A deep dive into the creation of my personal portfolio.

Project Overview

This project was my first major step into web development. The goal was to create a professional, responsive, multi-page website to act as a central hub for my skills, projects, and contact information. It was built entirely with foundational web technologies.

The Challenge

The main challenge was structuring the website logically across multiple pages and ensuring a consistent design and user experience. Implementing a fully responsive layout that works seamlessly on mobile, tablet, and desktop devices was a key learning objective.
